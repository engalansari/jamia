# Release Notes

## 1.0.0+1
- Initial Android release build.
- Arabic RTL interface.
- Firebase Authentication sign-in by username.
- Firestore-backed users, rounds, requests, logs, and notifications.
- Admin and regular user roles.
- Request creation, editing, deletion, and purchase flow.
- Basic English/LTR support with language switch.

## Known limitations
- iOS release requires macOS and Xcode.
- Final live phone tests require a stable connection to `firestore.googleapis.com`.
- Play Store upload requires a Google Play Console account and signing setup.
